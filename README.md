# E_Commerce_Bot
Project Description Developed a command-line based e-commerce recommendation bot using C++ that allows users to browse products by keywords, search items, and receive personalized product recommendations.

# Key Features
-Built a CLI-based system for user interaction and navigation
-Implemented keyword-based search to browse and filter products
-Developed product recommendation logic based on:
-User input keywords
-Product categories and attributes
-Designed structured data handling for product listings and search results
-Displayed recommended products in a clear and user-friendly format

# Tools & Concepts
-C++ (CLI-based application)
-Data Structures (arrays, vectors, or maps)
-Search & Filtering Algorithms
-Basic Recommendation Logic
